## What This Does

- **Learns** normal equipment behavior from historical data
- **Detects** anomalies in real-time sensor readings
- **Predicts** when equipment will fail
- **Alerts** maintenance teams before failure happens
- **Saves** companies ₹50,000 - ₹500,000+ per deployment

## How It Works

1. Train on 30 days of normal equipment data
2. Detect anomalies using statistical analysis (Z-score method)
3. Calculate degradation trends
4. Predict time-to-failure
5. Generate actionable alerts

## Use Cases

- Data center server failure prediction

- Power plant equipment monitoring
- Railway/aircraft maintenance
